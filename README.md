# Neon Flow

A cyberpunk-themed arcade shooter game built with vanilla JavaScript and DOM manipulation.

## Project Structure

```
Neon-Flow/
├── index.html          # Main HTML file
├── style.css           # Styles and animations
├── js/                 # JavaScript files
│   └── main.js         # Entry point
└── README.md           # This file
```

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- DOM Manipulation (No Canvas)

## Setup

1. Start a local server:
```bash
python3 -m http.server 8000
```

2. Open in browser:
```
http://localhost:8000
```
